# nota-release

배포 및 사용자 지원 전용 리포지토리입니다. 소스 코드는 포함되어 있지 않으며, 최신 설치 파일과 안내를 제공합니다.

This repository is dedicated to distribution and user support for nota. Source code is not included here; it serves download and support information only.

## Download / 다운로드
- GitHub Pages (Download): https://snowsee.github.io/nota-release/
- GitHub Releases: https://github.com/snowsee/nota-release/releases
- 최신 버전: https://github.com/snowsee/nota-release/releases/latest

## Support / 지원
- 질문이 있다면 GitHub Discussions를 이용해 주세요. (Questions? Please use GitHub Discussions.)
- Q&A: https://github.com/snowsee/nota-release/discussions

## Usage
- GitHub Pages는 `/docs` 디렉토리의 정적 페이지를 사용합니다. (Enable in Settings → Pages.)
- 다운로드 버튼과 링크는 항상 최신 GitHub Release를 참조합니다. (Links point to the latest releases.)

## Issues vs Discussions
- 설치/사용 질문과 제안은 Discussions를 사용합니다. (Use Discussions for questions and ideas.)
- 재현 가능한 버그 보고만 Issue에 등록합니다(활성화된 경우). (Use Issues only for reproducible bugs when enabled.)

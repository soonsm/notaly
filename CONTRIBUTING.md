# Contributing / 커뮤니티 참여

이 리포지토리는 nota의 배포 및 지원을 위한 공간입니다. 개발 관련 제안과 질문은 GitHub Discussions를 중심으로 운영합니다.

## Discussions 사용 목적 / Purpose
- 설치 및 사용 질문 (Installation and usage questions)
- 기능 제안 및 아이디어 (Feature suggestions and ideas)
- 일반적인 피드백 (General feedback)

## 권장 카테고리 구성 / Suggested Categories
- **Announcements**: 릴리스 공지 및 주요 안내 (Release notes and key announcements)
- **Q&A**: 질문/답변, Answer 지정 권장 (Questions/Answers; mark answers when resolved)
- **Ideas**: 기능 및 UX 제안 (Feature and UX proposals)
- **Known Issues**: 자주 보고되는 문제 (Frequently reported issues)
- **Show and Tell**: 사용 사례 공유 (Share how you use nota)

## Issues vs Discussions
- 질문 및 논의는 Discussions를 사용해주세요. (Use Discussions for questions or open-ended topics.)
- 재현 가능한 명확한 버그는 Issues를 사용합니다(활성화된 경우). (Use Issues only for reproducible, well-scoped bugs when enabled.)

## 운영 원칙 / Guidelines
- Q&A 스레드는 가능한 한 Answer를 지정합니다. (Mark answers in Q&A threads when resolved.)
- 중복 질문은 기존 스레드로 안내합니다. (Direct duplicates to existing threads.)
- 민감 정보 게시 금지: 로그, 개인정보, 토큰, 내부 경로 등은 공유하지 마세요. (Do not post sensitive information such as logs, personal data, tokens, or internal paths.)
- 공개 저장소이므로 검색 엔진에 노출될 수 있음을 유념하세요. (Remember Discussions are public and discoverable.)

Installation / 설치 방법

nota는 별도의 설정 없이 바로 설치해 사용할 수 있습니다.

(nota can be installed and used without additional configuration.)

  • Windows: 설치 파일 실행 후 안내에 따라 설치
  • macOS: 앱을 Applications 폴더로 이동
  • 최초 실행 시 OS 보안 경고가 나타날 수 있습니다.
